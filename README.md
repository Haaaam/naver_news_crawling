# naver_news_crawling
네이버 뉴스 기사 리스트 크롤링

### 사용 라이브러리
- BeautifulSoup
- requests
- pandas
- urllib

### 프로그램 소개
- find연산자 사용
- 크롤링 해오는 것: 뉴스 제목, 뉴스 내용 요약본
- 크롤링 결과: 딕셔너리-> 리스트 -> df -> csv and text로 저장
